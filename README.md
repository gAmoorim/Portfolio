# Gabriel Amorim - Portfolio

Portfolio pessoal desenvolvido com uma estética inspirada em sistemas Windows antigos, simulando uma experiência de desktop com janelas, atalhos, taskbar e navegação interativa.

## Sobre o Projeto

Este portfolio apresenta minhas principais informações profissionais, habilidades técnicas, projetos desenvolvidos e links de contato em uma interface visual diferente de um portfolio tradicional.

A proposta é unir identidade visual, interatividade e organização para destacar meus projetos como desenvolvedor.

## Tecnologias Utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- CSS

## Funcionalidades

- Tela de inicialização estilo sistema operacional
- Área de trabalho com atalhos
- Janelas interativas
- Botões de minimizar, maximizar e fechar
- Taskbar com janelas abertas
- Seção de projetos com preview, descrição, tecnologias e links
- Seção de skills
- Terminal visual com informações rápidas
- Área de contato
- Layout responsivo

## Projetos Apresentados

- Finance Flow
- LearnHUB
- DoIt
- Tech Store
- Noo Way Store
